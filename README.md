## 📖 Description

Saaki is the first project in the world to integrate a bipedal humanoid robot into a real operational environment within a hospital. This project will be carried out by our organization, Bioaraba, within the Innovation Support Unit (UAI). The robot will be deployed in the pediatrics department of the Araba University Hospital – Txagorritxu.

At the same time, we will continue researching the robot and, on our GitHub, we will publish code repositories and documentation about the Unitree G1. This will be the main repository, containing documentation in a "wiki" format and links to the rest of the repositories.

## 🎯 Project Goals

- 🏥 **Incorporation** of a bipedal humanoid robot into a healthcare environment
- 🔬 **Research** in AI and humanoid robotics applied to healthcare
- 🎓 **Training** students in advanced robotic technologies  
- 💡 **Innovation** in humanoid robot control and programming
- 🤝 **Interinstitutional collaboration** in technology projects
- 🤖 **Saaki** as a multiproject and research platform
