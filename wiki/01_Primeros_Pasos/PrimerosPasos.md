# Primeros Pasos y Recomendaciones

## 🖥️ Stack Tecnológico

Antes de desarrollar, se recomienda tener conocimiento y sentirse cómodo con las siguientes tecnologías:

- Linux (Ubuntu 22.04)
- Git y GitHub
- C++
- Python
- DDS
- ROS2
- Docker
- Entornos de simulación
- Computer Vision - Reinforcement Learning - Imitation Learning
- Desarrollo de APPS (web/móvil/escritorio)
- Robótica básica (actuadores, sensores...)

## 📋 Orden de Desarrollo

Recomendamos encarecidamente leer los documentos dentro de [esta carpeta](/wiki/02_Repositorios_UnitreeRobotics/), donde explicamos el ecosistema de desarrollo con las diferentes áreas sobre las que podemos trabajar y los diferentes repositorios que nos ofrece Unitree Robotics en su GitHub.

Una vez entendida la estructura y funcionamiento general del G1, se recomienda seguir un orden lógico para desarrollar:

1. SDK $\rightarrow$ Low Level
2. SDK $\rightarrow$ High Level
3. Acceso a cámara y sensor LiDAR
4. ROS2
5. Aprendizaje por Refuerzo o Imitación
6. VLA - WMA
