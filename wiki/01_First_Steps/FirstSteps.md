# First Steps and Recommendations

## 🖥️ Tech Stack

Before starting development, it is recommended to have knowledge and feel comfortable with the following technologies:

- Linux (Ubuntu 22.04)
- Git and GitHub
- C++
- Python
- DDS
- ROS2
- Docker
- Basic APP Development (web/mobile/desktop)
- Basic Robotics (actuators, sensors, SLAM, IK...)
- Simulation environments
- Artificial Inteligence:
  - Computer Vision
  - Large Language Models
  - Reinforcement Learning
  - Imitation Learning
  - Vision Language Action
  - World Model Action
  - ...

## 📋 Development Order

If you are new to the robot, we strongly recommend reading the documents within [this folder](/wiki/02_Repositorios_UnitreeRobotics/), where we explain the development ecosystem, the different areas we can work on, and the various repositories that Unitree Robotics offers on their GitHub.

Once the general structure and operation of the G1 are understood, it is recommended to follow a logical order for development:

1. SDK $\rightarrow$ Low Level
2. SDK $\rightarrow$ High Level
3. Camera and LiDAR sensor access
4. ROS2
5. Reinforcement or Imitation Learning
6. VLA - WMA
